# Dungeon-crawler
Python 3 dungeon crawler game with scripted content as well as randomly generated levels. It features a fully functional dice, character development, an ingame shop and randomly generated foes that drop treasure when defeated. You also get a highscore based on how many enemies you slay in your run.

This game is a great exercise for exploring Object-Oriented programming in Python. 

Prerequisites: 

Python 3.7+
random package
